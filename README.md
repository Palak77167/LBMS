# LBMS
Project Title - Library Book Management System
Project Description:
This project is a web-based Library Book Management System designed to automate the process of managing books, students, and borrowing records in a library.
The system reduces manual work, improves accuracy, and provides a simple interface for both Admin and Student users.

Purpose of the Project
The main purpose of this project is to:
Replace the traditional manual library system
Manage books digitally
Allow students to borrow and return books easily
Help admins maintain book records efficiently

Users / Roles
Admin
Add, update, and delete books
View all books in the library
Monitor borrowing activities

Student
View available books
Borrow and return books
View their own borrowing history
Technology Stack Used

Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT (JSON Web Token)
Authorization: Role-Based Access Control (RBAC)

Key Features

Secure login and registration
Role-based access (Admin / Student)
Book availability tracking
Book issue and return system
Borrow history for students
Real-time database updates

Security & Access Control
JWT-based authentication is used for secure login

RBAC ensures:
Only Admin can manage books
Students can access only their own records

Deployment
The project can be deployed on platforms like:
Render / Railway (Backend)
Netlify (Frontend)
